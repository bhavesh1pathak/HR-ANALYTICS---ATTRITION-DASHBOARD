# HR-ANALYTICS---ATTRITION-DASHBOARD
1️⃣ PROJECT TITLE / HEADLINE
HR Analytics Dashboard – Employee Attrition Analysis

2️⃣ SHORT DESCRIPTION / PURPOSE
Developed an interactive HR Analytics dashboard to analyze employee attrition patterns and identify key factors influencing employee turnover. The project helps HR teams make data-driven decisions to improve employee retention and workforce planning.

3️⃣ TECH STACK
Power BI – Dashboard creation & visualization
Power Query – Data cleaning and transformation (ETL)
DAX – KPI calculations and measures
Advanced Excel – Initial data exploration
Statistics – Trend and attrition analysis

4️⃣ DATA SOURCE
Kaggle – HR Analytics Employee Attrition Dataset

5️⃣ FEATURE HIGHLIGHTS
Designed KPI cards for Employee Count, Attrition Count, Attrition Rate, Average Age, Average Income, and Average Tenure
Analyzed attrition by Age Group, Gender, Education, Salary Slab, Department, Job Role, and Business Travel
Identified high attrition trends in early years of employment
Created interactive slicers for Department, Education Field, Age Group, and Business Travel
Provided actionable insights to reduce employee turnover and improve HR strategy
